Validate the YAML configuration file for SQA defined Schema. It uses SnakeYAML as YAML processor from Jackson package.

Fields:
  • yml: Mandatory YAML configuration .
  • schema: A Mandatory Object containing the data to be serialized.

Examples:

In the following code 'config.yml' and 'schema.json' represents the content of both files converted to String.


        def invalidMessages = indigoValidator yml: 'config.yml', schema: 'schema.json'
        if (invalidMessages) {
            error(invalidMessages.join('\n'))
        }
